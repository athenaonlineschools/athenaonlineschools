# 📚 About Athena Online School

Athena Online School is a modern EdTech platform designed to deliver high-quality online education through web and mobile applications.

The platform provides a complete educational ecosystem that enables:
- Students to attend live classes and track their progress
- Teachers to manage courses, lessons, and assignments
- Parents to monitor student performance and attendance
- Operators and admins to manage the educational system efficiently

Athena is built with scalability, performance, and modern cloud architecture in mind.
